Create an Express project which uses WebPack to create client files. Add this file to your static directory.

http://cs.harding.edu/gfoust/classes/comp431/email_list.html

Look at the HTML source. Notice it includes a file named jquery.js and a file named email_list.js, both from the /lib directory. These files will be created by WebPack. The file jquery.js is the commons chunk for jQuery, and the file email_list.js is the one you will write for this assignment.

You may not make any changes to the HTML file.

Problems

  1. When the user submits the form (either by clicking on the “Add” button or by pressing Enter) then you should add a new row to the end of the table with id users which contains the following three cells:
    • The value from the name text box
    • The value from the email text box
    • Three links (described below)
  2. The three links should be as follows:
    • An up-arrow link (&uarr;) with class="up"
    • A down-arrow link (&darr;) with class="down"
    • An delete link (X) with class="rem"
  3. When you click on the up-arrow link, the corresponding row should move up by one in the table unless it is already at the top.
  4. When you click on the down-arrow link, the corresponding row should move down by one in the table unless it is already at the bottom.
  5. When you click on the delete link, the corresponding row should be removed from the table.
